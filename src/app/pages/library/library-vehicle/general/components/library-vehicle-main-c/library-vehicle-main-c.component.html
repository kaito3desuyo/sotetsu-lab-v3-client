相模鉄道が所有する旅客用営業車両は、すべて20m・4扉に統一されており、2020年8月1日現在410両保有しています。過去には最大で442両を保有していた時期がありました。

現役の形式

新7000系

全60両製造 10両×3編成 30両現役 30両廃車

1975年から1989年にかけて製造された「7000系」車両のうち、1986年以降に製造された60両について一般的に「新7000系」と呼び、従前のそれとは区別されています。
従来の7000系（以下旧7000系）とは車体の外観が大きく変更されており、見た目は完全に別形式です。車内には停車駅案内装置が設置されています。3編成目からはVVVFインバータ制御となり、4編成目からは相鉄初の10両貫通編成となっています。最終編成は、一部車両が試験的にセミクロスシートとなり、その後の8000系・9000系に引き継がれています。
2006年5月20日のダイヤ改正で8連運用が増加したことに伴い、10連2編成が8連化されましたが、翌2007年5月から8月にかけて順次10連に戻されました。その後も一部編成がしばらく8連化されることがあり、2020年1月15日に7715×8が廃車されるまで残存しました。
2019年7月30日ごろの7713×10廃車を皮切りに老朽廃車が進行しており、12000系・20000系で完全に淘汰されるものと思われます。YNBへの塗装変更はいずれの編成も対象外です。

8000系

全130両製造 10両×11編成 110両現役 20両廃車

1990年から1999年にかけて、10連×13編成が製造され、「実質的に」相鉄車両の最大勢力となりました。「21世紀に通用する車両」として設計され、これまでの形式から様々な点が改善されています。
先頭車は、従来の切妻型デザインから、くの字に傾斜した現代的なデザインとされました。また、車体幅は新6000系（廃形式）以来の幅広構造となり、一人当たりのシート幅も430mmから450mmへ改善しています。その関係でドア間寸法が若干変化しています。さらに、側面に従来の形式からある列車種別に加え行先の表示もされるようになりました。
足回りは新7000系VVVF車とほぼ同一ですが、4M6Tから6M4Tに変更されており、パワーアップを果たしています。
製造期間が長かったことで製造年によって細部に差異があるほか、編成単位で細かい改良が施されているため、編成ごとにさまざまなバリエーションが見られます。
8707×10は、湘南台駅で発生した保線車両の暴走による衝突事故の影響で台枠が変形したことに伴い、2006年3月31日付で横浜方2両が廃車、残り8両も2006年12月に廃車されています。
2008年からは、一部編成の行先表示等がLEDタイプに更新されています。また、2015年2月ごろからはVVVFインバータ装置の更新工事が行われています。
2020年3月17日ごろに8702×10が廃車され、8000系初の老朽廃車となりました。12000系・20000系の導入に伴い8000系は8701×10から8708×10までの前期車が廃車されると予想されており、今後の動向が注目されています。後期車については、同年3月の8709×10のYNB化リニューアル竣工を皮切りに、順次YNB化を行うものと思われます。

9000系

全70両製造 10両×7編成 70両現役 廃車なし

1993年から2001年にかけて、10連×7編成が製造されました。8000系とは同時期製造ですが、製造企業が8000系は日立製作所なのに対し、9000系は東急車輛（現J-TREC 総合車両製作所）です。
先頭形状が8000系とは異なり、丸みを帯びたデザインとなっています。また、アルミ車ですが最初からカラー塗装されているのが特徴です。それ以外の8000系との違いとしては、車体幅が若干狭くなったこと、クーラーが集約分散式になったこと、最高速度が110km/hから120km/hに向上されたことなどがありますが、基本的には8000系と同様の車両と言えます。
9000系も8000系と同様の理由で、編成ごとに様々なバリエーションがあります。
2013年の9702×10を皮切りに、順次VVVFインバータ装置が東洋電機製GTO-VVVFから日立製IGBT-VVVFに更新されました。また、2015年から始まったYNB化リニューアルもこの9000系から行われており、何かと実験台になりがちな形式です。過去、9000系が新製された際に戦前製のPS13型パンタグラフが搭載され、相鉄ファンの度肝を抜いたことも記憶に新しいです。
8000系よりは若干新しい形式のため、12000系・20000系の導入による代替廃車は9701×10のみになると予想されています。9701×10以外の編成は、2019年10月の9707×10をもってすべてYNB化リニューアル竣工済みです。

10000系

全70両製造 10両×3編成 8両×5編成 70両現役 廃車なし

2001年（書類上は2002年）から2007年にかけて、8編成70両が製造されました。
JRのE231系と共通設計となっており、これにより大幅なコスト削減を図っています。10000系は同様にE231系と共通設計の東急5000系列と共に通勤・近郊電車の標準仕様ガイドライン策定のきっかけとなり、いわゆる「標準車両」の嚆矢となりました。
前面こそオリジナルデザインですが、車体構造や走行装置については、車体幅が車両限界の関係で2cm狭いこと以外はほぼE231系と同一です。また、相鉄では初めて車体にステンレスを採用した車両でもあります。
製造以来十数年程度、塗装変更を除けば大きな仕様変更なく走り続けてきましたが、2019年10月1日に機器更新のため10701×10がJR東日本の長野総合車両センターへと甲種輸送され、更新工事が施工されました。その後引き続きかしわ台車両センターでYNB化リニューアル工事を受けています。

11000系

全50両製造 10両×5編成 50両現役 廃車なし

2008年（書類上は2009年）から2013年にかけて、10両5編成が製造されました。
10000系に続き、JR東日本のE233系と共通設計を取っています。
当初2015年に開始する予定であったJR直通用に、各種準備工事を行ったうえで製造された車両でしたが、JR貨物横浜羽沢駅構内の工事の遅れに伴い、JR直通自体が4年程度遅れたため、相鉄デザインブランドアッププロジェクト策定後に製造された12000系にその役目を譲り、5編成製造した段階で製造が打ち切られました。
先代同様前面はオリジナルデザインですが、車体構造や走行装置はE233系とほぼ同様です。11000系導入前に車両限界がE233系と同じ2950mmに変更されたため、車体幅についても全くE233系と同様となっています。 全編成がYNB化リニューアル対象ですが、まだ施工を受けた編成はありません。
なお、この形式から車両の付番規則が変更されています。

12000系

全60両製造 10両×6編成 60両現役 廃車なし

2018年から2020年にかけて、10両6編成が製造されました。
延期されたJR直通の開始に合わせて製造され、相鉄デザインブランドアッププロジェクト策定後に製造された新車としては後述の20000系に続き2例目の登場となります。（20000系本格導入は12000系より後のため、順序が入れ替わっています）
JR車両と共通設計であることは先代・先々代と同様ですが、12000系の場合、車体構造はE235系と同じ「sustina S24」を採用している一方、走行装置については乗り入れ先の埼京線E233系7000番台と共通化を図るため、E233系と共通のものが採用されています。
前面は「獅子口」をイメージしたオリジナルデザインで、製造当初から車体はYNB一色に塗装されています。
再び付番規則が変更されています。（厳密には20000系から）

20000系

全40両製造 10両×4編成 40両現役 廃車なし

2018年から2020年にかけて、10両4編成が製造され、今後も増備される予定です。
12000系がJR直通用であるのに対し、こちらは東急に直通するための車両で、相鉄よりも狭い東急の車両限界に対応するため、旧6000系以来となる狭幅車体（日立A-trainベース）で製造されています。また、製造は8000系以来十数年ぶりに日立製作所笠戸事業所（山口県下松市）が担当し、20101×10甲種輸送の際にはEF66 30号機にけん引され大きな話題となりました。
20101×10は先行量産車的な位置づけであり、相鉄デザインブランドアッププロジェクト策定後の最初の新車として12000系に先んじて製造され、車内鏡の設置をはじめとした「相鉄らしさ」の演出や、デザイナーの手による意欲的な意匠が各所に凝らされていますが、その2年半後に製造された量産車である20102×10では、車内の枕木方向の液晶モニタがカット、ドア上モニタの筐体パネルの色変更等、一部で仕様変更が行われています。
東急直通用であるため、運転装置には相鉄では初めてT字ワンハンドルマスコンが採用されています。主制御装置に採用されたハイブリッドSiC-VVVFも当然ながら相鉄では初めての採用で、1C4Mの単独1M構成としており、将来目黒線直通の際に8連化が容易に行えるようになっています。
今後、東横線系統直通用の10両編成と、目黒線系統直通用の8両編成が順次製造され、新7000系・8000系前期車・9701×10を置き換える予定です。
